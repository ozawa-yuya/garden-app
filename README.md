# 🍀 家庭菜園作付シミュレーター

専門知識がなくても、スマホやPCから「野菜の最適な配置」を計算できるWebアプリを目指しています。

## 💡 このアプリでできること

- 農地と畝（うね）のサイズを入力すると、何本の畝が作れるか自動で計算します。
- 植えたい野菜の種類、株数と条数を入力すると、最適な配置を提案します。
- 野菜ごとの「株間」を考慮した配置計算を行い、スペースの無駄をなくします。

## 🚀 今後の展望（実装予定の機能）

知識がない人でも直感的に使えるよう、以下の順序でアップデートを予定しています。

1. **ビジュアル表示の強化**:
   野菜の配置結果を数字だけでなく、図面として一目で確認できるようにします（現在着手中のCSS描画機能）。
2. **マイ野菜登録機能**:
   ユーザーが自分で「トマト（株間50cm）」のように、育てたい野菜のデータを自由に追加・管理できるようにします。
3. **Web公開（デプロイ）**:
   現在は開発環境のみですが、最終的にはURLにアクセスするだけで誰でも使えるWebサービスとして公開します。

## 🛠 開発者向けセットアップ（技術情報）

1. **環境変数の設定**:
   - `DB_PASSWORD`: MySQL接続パスワード
2. **データベースの準備**:
   - 現在、データベースの構築用SQL（テーブル構造や初期野菜データ）については整理中のため、後日リポジトリにアップロード予定です。
   - 動作を確認される場合は、個別にテーブル作成が必要となります。ご了承ください。
3. **起動**:
   - `./gradlew bootRun`
